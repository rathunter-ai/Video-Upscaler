# OpenVideoUpscaler

Open source video upscaler written in Python.

Supports both traditional and AI-based upscaling with a clean, extensible architecture.

---

## Features

- Batch Processing
- FFmpeg Backend
- Lanczos Upscaling
- Real-ESRGAN
- NCNN Support
- Preserve Audio
- Multiple Video Formats
- Cross Platform
